\begin{abstract}
This paper investigates whether phonological learning primarily arises from the data representation (string vs. autosegmental representations) or from the learning model (probabilistic vs. categorical). We compare two learning frameworks: a Maximum Entropy (MaxEnt) model and BUFIA, a categorical learner adapted to operate over ARs. Using the same training and testing datasets across conditions, we systematically evaluate four combinations: MaxEnt-Ftr, BUFIA-Ftr, and BUFIA-AR. The results show that learning performance is strongest when grammars are induced over ARs, particularly when paired with a categorical learner. These findings suggest that representational structure plays a role in learning success than probabilistic weighting, and that categorical structure-sensitive learners benefit most from autosegmental representations.
\end{abstract}

\section{Introduction}

This paper focuses on the learning of tonal phonotactics, or tonotactics, which address the question of how tones and syllables (or other tone-bearing units, TBUs) are organized in a linguistically well-formed way in tonal languages. Despite the widespread distribution of tones in the world’s languages [cite], tonal phonotactics, as a core component of many languages, has not yet received the same level of attention as segmental phonology.

One possible reason for this gap may lie in representational differences. Segmental phonotactics is typically modeled using linear representations such as strings, which makes them readily applicable to a wide range of existing learning models. Tonotactics, on the other hand, has long been analyzed as depending on non-linear structures, such as autosegmental representations. This representational challenge has led to a relative scarcity of computational modeling of tones directly over non-linear structures, with many approaches instead flatten tonal forms into strings or features. Although there has been a great deal of work arguing for the inadequacy of linear representations in both the linguistic literature (for masking structural relations) and computational learning (where ARs may be more computationally efficient), the relevant works are still relatively limited. 

In this paper, we ask two questions: (1) how well tonotactics can be learned over ARs, and (2) how important representational differences are for learning compared with alternative string-based formats. To address this question, we adopt a recent version of the Bottom-Up Factorial Inference Algorithm (BUFIA) [cite], a categorical grammar induction algorithm that operates directly over ARs. We evaluate model performance and compare AR-based learning against alternative representations using the same algorithm as well as a Maximum Entropy (MaxEnt) learning algorithm [cite].

Specifically, we evaluate three conditions: string-based learning and AR-based learning, each learned by BUFIA-AR, or a probabilistic MaxEnt learner operating over strings. To ensure comparability, we convert the same Hausa dataset into two representations and use them as input to the models. We then compare model performance on held-out test data and constructed negative data. Admittedly, a complete study should also include MaxEnt operating over ARs. However, the current implementation of MaxEnt does not support learning directly from graphs. Despite this limitation, this comparison still allows us to assess whether a structural representation will contribute to learning.

The results show three main findings. First, tonotactics can be learned efficiently and effectively using BUFIA-AR, which deterministically returns a set of constraints consistent with the positive data. Some of the constraints learned by BUFIA align with previously established generalizations in the linguistic literature, while others reflect more specific conditions on the arrangement of tones and syllables. Second, we observe a learning difference between string-based and AR-based representations. When converted to ARs, the same BUFIA algorithm yields better recall and F1 scores, suggesting improved performance in recognizing positive data. Lastly, across models, categorical learning achieves better results than the probabilistic implementation in this task.

This work calls for greater attention to the use of linguistically meaningful representations in computational models of phonological learning.

In this section, we use Model Theory and view both representations as 
different mathematical objects. Different forms have distinct 
\textit{model signatures} that use formal logic to represent the 
relations of consisting units. 

A string can also be represented \textit{model-theoretically} 
with two components: the domain $D$ and a model signature of a finite 
set of relations $\mathcal{R}$. One can view $\mathcal{R}$ as a set of 
labeled nodes drawn from the alphabet, and a set of relations that capture 
the internal connections or ordering among those nodes. For example, a 
string model of \textit{L.HL} is shown in Figure~\ref{sec2.fig.string}.
Here, $D = \{1, 2, 3, 4\}$ indicates that there are four positions, 
among which $\{1, 4\}$ are labeled with $L$, $\{2\}$ with the boundary 
symbol $.$, and $\{3\}$ with $H$. These elements are connected via a 
binary successor relation, denoted by $\lhd$ in the signature, which 
specifies the linear order in which the symbols are pronounced.

\sigma, \mu\}$, where elements within each tier are in a successor 
relation, while elements across tiers stand in an association relation,

\subsection{Phonotactic Learning Models}
There are many string-based models that have been shown to be successful in 
phonotactic learning [cite]. One representative example is the Maximum 
Entropy (MaxEnt) phonotactic learner developed by [cite]. MaxEnt takes a 
set of positive data as input and returns a set of constraints assigned 
weights according to the principle of maximum entropy. It evaluates the 
well-formedness of words based on the sum of their weighted constraint 
violations. The higher the sum of constraint violateion, the less likely 
the candidate is considered as wellformed. 

Another approach, developed within formal language research, is categorical 
learning. Chandlee et al. \cite{chandleeLearningPartiallyOrdered2019} propose 
the Bottom-Up Factor Inference Algorithm (BUFIA), which discovers the most 
general forbidden subfactors represented in model-theoretic form and guarantees 
full coverage of the data space. BUFIA works by traversing a space of logically 
possible structures, moving from the most general (typically the empty 
structure) to more specific ones, and checking their presence in the positive 
data. If a structure is absent, BUFIA identifies it as a constraint of the 
language and prunes all of its superfactors, that is, structures that contain 
it. This procedure guarantees that the resulting grammar is the most general 
grammar consistent with the observed data.


\section{Experiment}
To test tonotactic learnability across representations and learning models, 
we conducted a controlled computational experiment on the same Hausa dataset 
using two representationas: linear string (or feature) representation
and AR representation. For the string representation, we implemented both 
\textsc{BUFIA} and \textsc{MaxEnt} learners, while for ARs, we implemented 
\textsc{BUFIA} only since the current \textsc{MaxEnt} implementation 
is not directly compatible with autosegmental structures without substantial 
configuration change.

consisting of six manually specified negative constraints established in the 
previous literature. This baseline allows us to evaluate whether the 
learning algorithms meaningfully improve over a linguistically informed,

\subsection{Evaluation}
We conducted a $4$-fold cross-validation procedure, partitioning the dataset 
into four equally sized folds. In each iteration, three folds are used for 
training and the remaining fold serves as the evaluation set. This process 
is repeated four times, with each fold acting as the test set once.

During evaluation, we do not test only on the held-out positive data. 
Instead, we construct two types of test sets: the \textit{illegal set} and 
the \textit{extended set}.

The \textit{illegal set} consists of the held-out positive forms (16 types) 
together with an equal number of manually constructed negative forms. 
Each constructed negative form violates at least one of the baseline 
constraints. For each condition, the test items are encoded in the 
corresponding representation (string or AR) and evaluated together with 
the positive data. This design allows us to assess whether the learned 
grammar over generates beyond the attested patterns.

The \textit{extended set} builds on the illegal set by adding four 
exceptional positive types and four matched negative forms (40 in total). 
The exceptional forms are drawn from Hausa data 
reported by Newman and violate both \textsc{*Clash} and 
\textsc{*Nonfinal-Fall}. Newman notes that a few such forms do occur in the 
corpus and suggests that they may have historical explanations, although 
their precise source remains unclear. 

In model evaluation, a well-formedness judgment is counted as correct if the 
model accepts positive examples (whether from the held-out fold or the 
exceptional set) and rejects the negative examples. Performance is 
measured in terms of recall, precision, and F1 score over these 
classifications.

It should be noted that the \textsc{MaxEnt} model returns a gradient 
grammar in which all constraints are violable and assigned weights. 
However, for comparability with the categorical models, we 
convert the learned  \textsc{MaxEnt}  grammar into a categorical 
interpretation: Candidate forms whose total weighted violation score 
is zero are treated as well-formed, whereas those with a non-zero 
weighted sum are treated as ill-formed. In other words, a test candidate 
is classified as well-formed only if it does not violate any constraint 
in the grammar.





\section{Results}
The averaged results across folds are shown in Tables~4 and~5. For both 
test sets, all models under both representational formats achieved perfect 
precision (1.000). This indicates that, regardless of the learning 
mechanism, both string and AR representations successfully reject the 
constructed ill-formed structures.

Recall and F1 scores, however, show greater variation. Across both test 
sets, the same performance pattern emerges:

\textsc{BUFIA-AR} > \textsc{BUFIA-string} > 
\text{baseline-string} = \text{baseline-AR} > 
\textsc{MaxEnt-string}.

This ranking is consistent across both the held-out data and the extended 
test set including exceptional forms.

First, the baseline conditions yield identical results across 
representations. This indicates that for linguistically informed, manually 
specified grammars operating over surface forms, the choice of data 
representation does not affect prediction. This is expected, as the string 
and AR datasets were constructed in a one-to-one corresponding manner.

In contrast, the choice of representation substantially affects the 
learning models. Specifically, \textsc{BUFIA-string} achieves a recall of 
0.906 under the string representation, but this increases to 0.984 under 
the AR representation. The improvement in performance suggests 
that representational choice plays a role in learnability.
